Paradise Nursery Shopping App

Paradise Nursery is a front-end e-commerce shopping cart application developed as part of the IBM Front-End Development course. The application allows users to browse houseplants, add products to a cart, update quantities, remove items, and view real-time cart totals.

The project includes a landing page, product listing page, and shopping cart page. It demonstrates practical React development skills including component-based architecture, conditional rendering, navigation, reusable UI components, and Redux-based global state management.

Key Features
React-based single-page application
Landing page with company branding and call-to-action
Product listing page with categorized houseplants
Dynamic shopping cart icon showing total item count
Add-to-cart functionality with disabled buttons after selection
Shopping cart page with item quantity controls
Increase, decrease, and delete cart item functionality
Real-time calculation of total items and total cost
Redux slice for managing cart state
GitHub Pages deployment
Skills Demonstrated
React component development
Redux state management
JavaScript logic for cart operations
UI routing and navigation
Dynamic rendering of product and cart data
CSS styling and responsive layout structure
GitHub repository management
Front-end deployment using GitHub Pages

This project demonstrates my ability to build interactive front-end applications using React and Redux, manage application state cleanly, and structure a project for real-world deployment. The app follows the IBM project requirements for a shopping interface with landing, product listing, and cart functionality.
