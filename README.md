# Paradise Nursery Shopping App

Paradise Nursery is a front-end e-commerce shopping cart application developed as part of the IBM Front-End Development course.

The application allows users to browse a selection of houseplants, add products to a shopping cart, update item quantities, remove items, and view the total cart cost in real time.

## Project Overview

This project includes three main pages:

- Landing Page
- Product Listing Page
- Shopping Cart Page

The goal of the project was to build a React-based shopping interface that demonstrates practical front-end development skills, including reusable components, navigation, state management, and dynamic UI updates.

## Key Features

- React single-page application
- Landing page with company branding and a Get Started button
- Product listing page with houseplants grouped into categories
- Shopping cart icon with dynamic item count
- Add-to-cart functionality
- Disabled Add to Cart button after a product is selected
- Shopping cart page displaying selected plants
- Increase and decrease quantity controls
- Delete item functionality
- Real-time total item and cost calculation
- Continue Shopping button
- Checkout button with a Coming Soon message
- GitHub Pages deployment
